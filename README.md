# Welding Services Website

## About

**ARGON66** is a modern one-page website built with **React** for a welding services company.

The project features a minimalist **black-and-white design**, responsive layout, and an integrated contact form powered by **EmailJS**, allowing customers to quickly submit service requests.

---

## Design

- Black & White color palette
- Minimalist industrial style
- Responsive layout
- Smooth scrolling navigation
- Modern UI
